CSS

CSS (Cascading Style Sheets) is what brings the visual appeal to a website. It allows me to style and design the web pages, ensuring they are attractive and user-friendly. JavaScript

JavaScript is a versatile programming language crucial for adding interactivity and dynamic features to web pages. I use TypeScript to write robust and scalable code, ensuring maintainability and reducing the likelihood of bugs in large codebases.
